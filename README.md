# 🔥 Pranjal Free Fire - Custom Game Engine

**Premium Battle Royale & Competitive FPS Game**

## 📋 Project Overview

Pranjal Free Fire is a high-performance mobile game engine built from scratch with two primary game modes:

### 🎮 Game Modes
1. **Battle Royale (50v50)** - Last Person Standing
   - Dynamic Shrinking Zone System
   - Full Looting Mechanics
   - Real-time Multiplayer
   - Ranking System

2. **CS Ranked (4v4)** - Competitive Team-Based
   - Tactical Gameplay
   - Skill-based Ranking
   - Real-time Match Statistics
   - Tournament Support

## 🛠️ Tech Stack

- **Game Engine**: Custom C++ Engine
- **Backend**: Firebase (Real-time Database, Authentication, Cloud Functions)
- **Platforms**: Android (Java/Kotlin) + iOS (Swift)
- **Graphics**: Custom Rendering Pipeline with OpenGL ES 3.0
- **Networking**: WebSocket + Firebase Realtime Database
- **Physics**: Custom 3D Physics Engine
- **Audio**: OpenAL Audio System
